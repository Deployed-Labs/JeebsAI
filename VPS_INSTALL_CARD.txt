🎯 TO INSTALL ON YOUR VPS:

   Option 1 - Direct Install (Easiest):
   ────────────────────────────────────
   curl -sSL https://raw.githubusercontent.com/Deployed-Labs/JeebsAI/main/vps_fresh_install.sh | sudo bash

   Option 2 - Download & Run:
   ────────────────────────────
   wget https://raw.githubusercontent.com/Deployed-Labs/JeebsAI/main/vps_fresh_install.sh
   chmod +x vps_fresh_install.sh
   sudo ./vps_fresh_install.sh

═══════════════════════════════════════════════════════════════

✅ WHAT IT DOES:
   • Installs all dependencies (Rust, build tools, etc.)
   • Clones JeebsAI repository to /root/JeebsAI
   • Builds release binary (~5-10 minutes)
   • Creates systemd service
   • Starts JeebsAI automatically
   • Performs health check

⏱️  INSTALL TIME: 5-15 minutes (depending on VPS speed)

📋 AFTER INSTALLATION:

   Access JeebsAI:
   http://YOUR_VPS_IP:8080

   Check Status:
   sudo systemctl status jeebs

   View Logs:
   sudo journalctl -u jeebs -f

   Update Later:
   cd /root/JeebsAI && sudo ./deploy_to_vps.sh
